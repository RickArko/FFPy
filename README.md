# FFPy
Fantasy Football & Python

## Getting Started

### Pushing Your Existing Repository

If you have an existing local repository and want to push it to this GitHub repository, see the [Setup Guide](SETUP.md) for detailed instructions.

**Quick Start:**
```bash
cd /path/to/your/local/repository
git remote add origin https://github.com/RickArko/FFPy.git
git fetch origin
git merge origin/main --allow-unrelated-histories
git push -u origin main
```

For more detailed instructions, including SSH setup, troubleshooting, and alternative methods, please refer to [SETUP.md](SETUP.md).
